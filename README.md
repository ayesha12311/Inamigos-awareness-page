# InAmigos Foundation — NGO Awareness Webpage

A modern, responsive awareness website created to showcase the mission, initiatives, impact, and community outreach efforts of InAmigos Foundation.

## Overview

This project was developed as an NGO awareness webpage to promote the work of InAmigos Foundation and encourage public participation through volunteering, awareness, and community engagement.

The website highlights the foundation's mission, key projects, social impact, leadership, and gallery of activities while maintaining a clean, accessible, and responsive user experience.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Modern NGO-focused user interface
* Light and dark mode support
* Interactive navigation bar
* Hero section with call-to-action
* About section introducing the foundation and founder
* Project showcase section
* Impact and mission highlights
* Community activity gallery
* Volunteer and participation call-to-action section
* Animated statistics counter
* Smooth scrolling navigation
* Mobile-friendly navigation menu

## Sections Included

### Home

Introduces the foundation's mission and vision with prominent calls-to-action.

### About InAmigos Foundation

Provides information about the organization, its founding, registrations, certifications, and leadership.

### Projects

Highlights the foundation's major initiatives:

* Project Seva (Community Welfare)
* Project Bachpanshala (Education)
* Project Jeev (Animal Welfare)
* Project Udaan (Women Empowerment)
* Project Prakriti (Environment)
* Project Vikas (Skill Development)

### Impact

Explains the social importance and long-term goals of the foundation's initiatives.

### Gallery

Displays photographs of community outreach programs, volunteer activities, and welfare initiatives.

### Join Us

Encourages visitors to volunteer, support, and participate in social impact activities.
